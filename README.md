# JDBC CRUD Application

A simple Java-based CRUD (Create, Read, Update, Delete) application developed using JDBC and MySQL. This project demonstrates database connectivity, SQL operations, and Java programming concepts through a console-based application.

## 🚀 Features
- Create records
- Read records
- Update records
- Delete records
- JDBC & MySQL integration

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL
- SQL
- Eclipse IDE
- Git & GitHub

## ▶️ How to Run
1. Clone the repository.
2. Configure your MySQL database.
3. Update database credentials in the Java file.
4. Run the application.

## 📚 Learning Outcomes
- JDBC Connectivity
- SQL CRUD Operations
- Database Management
- Exception Handling
- Git & GitHub Workflow
